# 🏆 Rock Paper Scissors (Vanilla JavaScript)

A lightweight and interactive Rock–Paper–Scissors game built using **HTML**, **CSS**, and **Vanilla JavaScript**. The application allows users to play against the computer, track scores, and persist results using browser storage.

---

## 🎯 Overview

This project demonstrates fundamental front-end development concepts, including:

- DOM manipulation  
- Event handling  
- Game logic implementation  
- Local storage usage for data persistence  

The interface is minimal and responsive, focusing on usability and simplicity.

---

## ✨ Features

- Interactive UI with emoji-based controls 👊🏻 🤚🏻 ✌🏻  
- Real-time game result display  
- Persistent score tracking using `localStorage`  
- Reset functionality to clear scores 🔄  
- Simple and responsive layout  

## ⚙️ How It Works

1. The user selects one of the three options:
   - Rock (👊🏻)
   - Paper (🤚🏻)
   - Scissors (✌🏻)

2. The computer randomly generates its choice.

3. The game logic compares both selections and determines:
   - Win  
   - Loss  
   - Tie  

4. The result and updated score are displayed dynamically.

## 🚀 Installation & Usage

1. Clone or download the repository:
```bash
git clone https://github.com/your-username/rock-paper-scissors-js.git
```
2. Navigate to the project directory:
```bash
cd rock-paper-scissors-js
```
3. Open index.html in any modern web browser and enjoy.
